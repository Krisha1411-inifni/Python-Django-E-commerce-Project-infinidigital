# 🛒 Django E-commerce Project

This is a full-stack e-commerce web application built using Django.

---

## 🚀 Features
- User authentication (Login/Register)
- Product listing
- Shopping cart
- Order management
- Admin panel

---

## 🛠️ Tech Stack
- Python
- Django
- SQLite
- HTML, CSS, JavaScript

---

## ⚙️ Setup Instructions

1. Clone the repository:
git clone https://github.com/Krisha1411-inifni/Python-Django-E-commerce-Project-infinidigital.git

2. Navigate to project:
cd Python-Django-E-commerce-Project-infinidigital

3. Create virtual environment:
python -m venv venv

4. Activate environment:
venv\Scripts\activate

5. Install dependencies:
pip install -r requirements.txt

6. Run server:
python manage.py runserver
